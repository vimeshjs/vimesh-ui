按钮API参考

属性 可选值 默认值 说明
data-type default, primary, secondary, accent, info, success, warning, error, ghost, link default 按钮类型
data-size xs, sm, md, lg, wide md 按钮尺寸
data-shape default, square, circle, block default 按钮形状
data-outline true, false false 是否为轮廓按钮
data-glass true, false false 是否启用玻璃效果
data-loading true, false false 是否显示加载状态
data-disabled true, false false 是否禁用按钮
data-noAnimation true, false false 是否禁用动画效果
